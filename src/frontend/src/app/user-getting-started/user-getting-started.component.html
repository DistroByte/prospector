Welcome to Prospector

What will I be able to do in Prospector?

Here is where you will be able to deploy your own projects. You are given an environment with a set quota of resources to use. From then you can deploy a webserver or a database and any docker or approved virtual machine.

Lets get started by creating your first Project in Prospector

Your project name should be unique to avoid any confusion in your environment. For now lets put Hello_World!

Projects can be made up of docker container(s) and virtual machines. For now lets focus on docker containers.

You can have multiple components in a docker project, and can select various options such as what port to use or to expose your component, etc.
